# 📱 Projeto PoolPhone

Este projeto foi desenvolvido em **Java** e simula as principais funções de um celular, contendo:

- 🎵 **Reprodutor Musical**  
- 📞 **Aparelho Telefônico**  
- 🌐 **Navegador de Internet**
